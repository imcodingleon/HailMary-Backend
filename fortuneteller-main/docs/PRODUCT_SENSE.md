# Product Sense

> 제품 감각, 사용자 가치, 도메인 지식

## 제품 정의

한국 전통 사주팔자(四柱八字) 분석을 AI 어시스턴트에게 제공하는 MCP 도구 서버.
사용자가 AI 대화 속에서 자연어로 사주 분석을 요청하면, 정확한 계산과 전문적 해석을 제공한다.

## 핵심 사용자 가치

### 1. 정확성

- KASI(한국천문연구원) 앵커 데이터 기반 검증
- 진태양시 보정 (162개 시군구 경도 + 썸머타임)
- 300년 범위 절기/음력 데이터 내장

### 2. 전문성

- 10단계 사주 계산 파이프라인 (명리학 이론 기반)
- 5개 해석 유파 비교 (자평명리, 현대명리, 적천수, 궁통보감, 신살중심)
- 4종 용신 알고리즘 (강약/조후/통관/병약)

### 3. 접근성

- MCP 프로토콜로 AI 어시스턴트와 자연어 대화 통합
- 복잡한 입력을 AI가 대화로 수집 -> 도구 호출
- 전문 용어를 AI가 일상 언어로 풀이

## 도메인 지식 요약

### 사주팔자란?

생년/월/일/시의 4개 기둥(四柱), 각 기둥의 천간+지지 2글자씩 총 8글자(八字)로 개인의 운명 구조를 분석하는 동양 전통 학문.

### 핵심 개념

| 개념 | 설명 | 프로젝트 매핑 |
|------|------|---------------|
| 천간(天干) | 10개 (갑을병정무기경신임계) | `heavenly_stems.ts` |
| 지지(地支) | 12개 (자축인묘진사오미신유술해) | `earthly_branches.ts` |
| 오행(五行) | 목화토금수 - 상생/상극 관계 | `wuxing.ts` |
| 십성(十星) | 일간 기준 10개 관계 유형 | `ten_gods.ts` |
| 격국(格局) | 사주 구조 분류 | `gyeok_guk.ts` |
| 용신(用神) | 균형을 위한 유리한 오행 | `yongsin/` |
| 대운(大運) | 10년 단위 운세 주기 | `dae_un.ts` |

### 사용자 인터랙션 패턴

1. 사용자가 사주 분석 요청
2. AI가 필수 정보 수집 (이름, 생년월일시, 양음력, 성별, 출생지)
3. AI가 MCP 도구 호출
4. 서버가 10단계 파이프라인으로 사주 계산
5. AI가 결과를 6항목 해석 프레임워크로 풀이

## 경쟁 우위

- **오프라인 동작**: 외부 API 의존 없음 (인터넷 없이도 동작)
- **유파 비교**: 5개 유파 동시 비교 기능
- **AI 통합**: MCP 프로토콜로 자연어 대화 속 자연스러운 분석
- **진태양시**: 출생지 경도 보정으로 시주 정확도 향상
