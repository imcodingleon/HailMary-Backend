# 명리 해석 가이드

> 사주 분석 결과를 사용자에게 풀이할 때 따르는 규칙

## 사주팔자 조회 전 사용자 확인

**사주 도구 호출 전에** 사용자에게 아래를 반드시 확인:

| 항목 | 설명 |
|------|------|
| 이름(한글) | 호칭/기록용 |
| 한자 이름 | 선택. 모르면 생략 가능 |
| 양력/음력 | `calendar`: solar / lunar |
| 생년월일/시각 | `birthDate` YYYY-MM-DD, `birthTime` HH:mm |
| 음력 윤달 | 음력일 때만 `isLeapMonth` |
| 성별 | `gender`: male / female |
| 태어난 곳 | 시군구 한글명 (`birthCity`). 생략 시 서울 |

이름/한자는 대화 맥락용. 도구 호출에는 스키마 필드만 전달.

## 명리 해석 6항목

`analyze_saju` 결과 기반, 명리학 전문가 관점에서 상세 서술:

1. **일간/일주 중심** - 본연의 기질과 중심 성격
2. **월지 기운/십성 흐름** - 사회적 환경과 역량 발휘 방식
3. **십성 구성의 장단점/인생 흐름** - 특징적 장단점과 인생 흐름
4. **오행 분포/실생활 보완** - `wuxingCount` 기반 보완책 (색상, 습관 등)
5. **주요 영역 종합** - 재물운, 연애/결혼운, 직업 적성, 건강운
6. **균형/삶의 태도** - 사주 균형을 위한 핵심 조언

**원칙**: 도구가 반환한 필드만 근거로 삼고, 없는 내용을 지어내지 않는다.

## 엔진/자동 해석과의 충돌 방지

| 이슈 | 규칙 |
|------|------|
| `wuxingCount` | 사기둥 천간/지지 오행 개수 합. 다른 학파 무게 체계와 다를 수 있음을 한 줄 언급 |
| `fortune` 문장 | 엔진 템플릿 문장과 겹치면 6항목 순으로 재구성 또는 인용 후 심화 해석 |
| 용신/격국 | 엔진의 `yongSin`/`gyeokGuk`과 모순 금지. 유파 차이는 `school_compare` 참고 |
| 십성 분포 | 반환 JSON의 십성 필드 우선. 다른 이론서 정의와 다를 수 있음 |
